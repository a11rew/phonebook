[https://phonebook-pi.vercel.app/](https://phonebook-pi.vercel.app/)

### Development

After cloning, you'll need to install the dependencies:

```
$ cd phonebook
$ yarn
```

Then, start the development server on port 3000:

```
$ yarn dev
```
